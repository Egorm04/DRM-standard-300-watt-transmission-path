# DRM-standard-300-watt-transmission-path

# Передающий тракт DRM+ мощностью 300 Вт

**Стандарт:** DRM+ (Digital Radio Mondiale, Band II)  
**Диапазон частот:** 87.5 – 108 МГц  
**Выходная мощность (пиковая):** 300 Вт  
**Волновое сопротивление антенны:** 50 Ом  
**Статус:** проектирование завершено

---

## Обзор

Проект содержит полную документацию по разработке передающего тракта для системы DRM+.  
DRM+ – цифровая система радиовещания, использующая OFDM и QAM модуляцию (4‑QAM, 16‑QAM), предназначенная для диапазона ОВЧ (FM‑диапазон).

В рамках проекта выполнены:

- анализ технических требований (спектральная маска, стабильность частоты, качество сигнала);
- выбор и обоснование структурной схемы (без переноса частоты);
- расчёт маломощного тракта: опорный генератор, буферный усилитель, синтезатор частоты (DDS), квадратурный модулятор;
- проектирование усилителя мощности: предварительные каскады (MRF158, MRF134) и оконечный каскад (BLP05H9S500);
- расчёт выходной согласующей П‑цепи и фильтра нижних частот для подавления гармоник.

Все расчёты выполнены с учётом реальных параметров компонентов (даташиты). Приведены номиналы, режимы работы, КПД, коэффициенты усиления.

---

## Ключевые результаты

| Параметр | Значение |
|----------|----------|
| Выходная мощность (пик) | 421 Вт (с запасом 20%) |
| КПД оконечного каскада | 64.8 % |
| Коэффициент усиления всей тракта | ~ 52 дБ |
| Нестабильность частоты | 1 ppm (TCXO) |
| Шаг перестройки частоты | 0.11 Гц (синтезатор AD9910) |
| Подавление второй гармоники фильтром | > 43.6 дБ |
