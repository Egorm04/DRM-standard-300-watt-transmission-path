# Номиналы компонентов передающего тракта DRM+ 300 Вт

## Опорный генератор и буфер

| Компонент | Тип / модель | Номинал / параметр |
|-----------|--------------|---------------------|
| X1 | TCXO | ECS-TXO-25CSMV-400-BN-TR, 40 МГц |
| U2 | буферный усилитель | BUF634P |
| R1, R2 | резисторы | 100 Ом (по даташиту) |
| C1, C2 | конденсаторы фильтра питания | 0.1 мкФ, 10 мкФ |

## Синтезатор AD9910

| Выводы | Назначение | Компоненты |
|--------|------------|------------|
| REF_CLK (90,91) | вход опоры 40 МГц | дифференциальная цепь с 1:1 трансформатором |
| PLL_FILTER (2) | внешний фильтр | R1=51 Ом, C1=130 нФ, C2=27 нФ |
| DVDD (1.8 В) | цифровое ядро | феррит + 0.1 мкФ + 10 мкФ на каждый пин |
| AVDD (3.3 В) | питание ЦАП | феррит + 0.1 мкФ + 10 мкФ |
| IOUT (75,76) | выход DDS | нагрузка 50 Ом на землю через трансформатор |
