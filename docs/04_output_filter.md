# Выходной фильтр и согласующая цепь

## Согласование оконечного каскада с нагрузкой 50 Ом

R_экв оконечного каскада = 10.4 Ом (для одного плеча). Для двух плечей, работающих на суммирующий трансформатор, эквивалентное сопротивление нагрузки каждого транзистора – 25.4 Ом (при симметричном сложении). Требуется согласовать 25.4 Ом → 50 Ом.

Используется **П-образная согласующая цепь**, выполняющая также предварительную фильтрацию гармоник.

**Расчёт цепи (f₀ = 98 МГц):**

| Параметр | Формула | Значение |
|----------|---------|----------|
| R_экв | 25.4 Ом | – |
| R | 50 Ом | – |
| R₀ | (R_экв·R)/(R_экв+R) | 16.84 Ом |
| C₁ | √(R_экв/R₀ – 1)/(2πf₀·R_экв) | 45.6 пФ |
| C₂ | √(R/R₀ – 1)/(2πf₀·R) | 45.6 пФ |
| L | R₀(√(R_экв/R₀-1)+√(R/R₀-1))/(2πf₀) | 57.9 нГн |

**Рабочие напряжения и реактивная мощность:**

- U_C1 ≈ 230 В, P_C1 ≈ 1.63 квар
- U_C2 ≈ 230 В, P_C2 ≈ 2.57 квар

**Ослабление согласующей цепи:** a_сц = –11.6 дБ.

## Фильтр нижних частот (подавление гармоник)

Требуемое подавление второй гармоники (f₂ = 196 МГц) с учётом коэффициентов Берга:

a_г2 = 20·log(α₁/α₂) = 20·log(0.54/0.09) = 15.6 дБ (в нашем расчёте 14.65 дБ – разница из-за округления).

Необходимое подавление фильтром (при требовании спектральной маски –70 дБ на сдвиге 100 кГц):

a_ф ≥ –a_доп + a_г2 – a_сц = 70 – 14.65 – 11.6 = 43.75 дБ.

**Спроектированный ФНЧ** – эллиптический 5-го порядка (LC).  
Расчёт производился методом синтеза по прототипу с частотой среза 108 МГц.

*Схема фильтра:* (см. рисунок 18 в оригинале)  
Номиналы: L1 = 72 нГн, C1 = 33 пФ, L2 = 110 нГн, C2 = 22 пФ, C3 = 18 пФ.

**Результат моделирования:**  
- Ослабление на 196 МГц > 45 дБ.  
- Неравномерность в полосе пропускания < 0.1 дБ.

## Итоговая схема выходной части

Согласующая П-цепь → ФНЧ → антенна (50 Ом).
