# Системные требования

## Основные параметры

| Параметр | Значение |
|----------|----------|
| Стандарт | DRM+ (ETSI TS 102 980) |
| Виды модуляции | 4-QAM, 16-QAM (OFDM) |
| Диапазон частот | 87.5 – 108 МГц (Band II) |
| Номинальная выходная мощность | 300 Вт (средняя) |
| Пиковая мощность (с учётом запаса) | 421 Вт |
| Волновое сопротивление антенны | 50 Ом |
| КСВН антенны | ≤ 1.1 |
| Тип передатчика | стационарный |

## Электромагнитная совместимость

Внеполосная спектральная маска (канал 100 кГц):

| Сдвиг частоты (кГц) | Уровень (дБн/1 кГц) |
|---------------------|---------------------|
| 0 … ±50 | –20 |
| ±70 | –50 |
| ±100 | –70 |
| ±200 | –80 |
| ±300 | –85 |
| ≥ ±400 | –85 |

*Графическое представление маски – в приложении.*

**Нестабильность частоты:** ±2×10⁻⁶ (для диапазона до 160 МГц).  
Фактически используется TCXO с нестабильностью 1×10⁻⁶.

## Качество сигнала

- Измерение BER: диапазон 10⁻⁶ … 10⁻³
- Измерение MER: до 40 дБ, точность ±1 дБ

## Эксплуатационные требования

- Напряжение питания: 220 В (50 Гц)
- Диапазон рабочих температур: –20 … +50 °C
- Охлаждение: принудительное (вентилятор) для оконечного каскада

## Используемые нормативные документы

- Решение ГКРЧ от 11.09.2018 № 18-46-01 (внедрение DRM+ в России в диапазонах 65.9–74 МГц и 87.5–108 МГц)
- Стандарт ETSI TS 102 980 (DRM+)
