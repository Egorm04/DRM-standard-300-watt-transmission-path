# Усилитель мощности

## Исходные данные

- Выходная мощность модулятора: +3 дБм (2 мВт)
- Требуемая пиковая мощность: 421 Вт (с учётом КПД фильтра и запаса)
- Антенна: R = 50 Ом, КСВН = 1.1

Расчёт потерь:

P_вых = 300 Вт  
K_бв = 0.9 → P_вфс = 300 / √0.9 = 316.23 Вт  
P_ок = 316.23 / 0.9 = 351.36 Вт  
С запасом 20%: P_ок_треб = 421 Вт

## Выбор режима работы

**Класс AB** (угол отсечки θ = 120°) – оптимален для сигналов с переменной огибающей (OFDM).  
Режим по напряжённости: **слегка перенапряжённый** (ζ = 0.98) для высокого КПД.

## Оконечный каскад

**Транзистор:** BLP05H9S500 (NXP) – LDMOS, 500 Вт, 50 В, диапазон до 1 ГГц.

**Расчёт (при P_ок = 195 Вт на транзистор, т.к. используется два транзистора в мостовой схеме):**

| Параметр | Формула | Значение |
|----------|---------|----------|
| ζ | 0.5+0.5√(1-8P_ок r_нас/(α₁E_c²)) | 0.981 |
| U_с | ζ·E_c | 63.8 В |
| I_c1 | 2P_ок / U_с | 6.11 А |
| i_c_max | I_c1/α₁ | 11.3 А |
| I_c0 | α₀·i_c_max | 4.63 А |
| R_экв | U_с / I_c1 | 10.4 Ом |
| P_0 | E_c·I_c0 | 301 Вт |
| η | P_ок/P_0 | 64.8 % |
| P_вх (требуемая) | – | 1.9 Вт |

**Коэффициент усиления по мощности:** ~ 44 дБ (от входа оконечного каскада до выхода).

## Предоконечный каскад

**Транзистор:** MRF134 (Motorola) – 3 Вт, 28 В.

- Выходная мощность: 1.9 Вт (необходима для раскачки оконечного)
- Входная мощность, требуемая для MRF134: 0.21 Вт
- Коэффициент усиления: 12.3 дБ

## Дополнительный каскад предварительного усиления

**Транзистор:** MRF158 (Motorola) – 1 Вт, 28 В.

- Выходная мощность: 0.21 Вт
- Входная мощность: 1.19 мВт
- Коэффициент усиления: 22.5 дБ

Так как выход модулятора – 2 мВт, этого достаточно для раскачки MRF158.

**Итоговая структура усилителя:**

Модулятор (2 мВт) → MRF158 (1.19 мВт вх., 0.21 Вт вых.) → MRF134 (0.21 Вт вх., 1.9 Вт вых.) → BLP05H9S500 (1.9 Вт вх., 195 Вт вых.) → суммирование двух плечей → 390 Вт → после согласования и фильтра → 300 Вт.
